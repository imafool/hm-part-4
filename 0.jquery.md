## jquery
```css
1. 是一个 JavaScript 函数库
2. 包含以下特性：
   1. HTML 元素选取 
   2. HTML 元素操作 
   3. CSS 操作 
   4. HTML 事件函数 
   5. JavaScript 特效和动画 
   6. HTML DOM 遍历和修改 
   7. AJAX 
   8. Utilities
```

## 引入
```html
<head>
<script type="text/javascript" src="jquery.js"></script>
</head>

<head>
  <script type="text/javascript" src="http://ajax.microsoft.com/ajax/jquery/jquery-1.4.min.js"></script>
</head>
```
